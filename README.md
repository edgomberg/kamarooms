# KamaRooms /open

The world's first AI-native hotel transparency dashboard. Real metrics, real story, real data. Published openly.

## The Story

Hotel in Naberezhnye Chelny, Russia. 108 rooms. Burned in October 2023. 72 people evacuated in 7 minutes, zero casualties. Full reconstruction at 90M+ RUB. Record revenue in 2025.

Then we decided to publish everything.

## Features

- **Bilingual.** Full EN/RU toggle on every page.
- **Real data.** 2025 metrics from the hotel's PMS and financial systems.
- **Interactive charts.** Chart.js with tabs and time range selectors.
- **Brand typography.** Stolzl font family, sourced from kamarooms.com.
- **Static HTML.** No framework. No build step. No dependencies at runtime.

## Tech Stack

- Static HTML + CSS + vanilla JS
- Chart.js for interactive charts
- Stolzl webfont (Tilda CDN)
- GitHub Pages hosting
- No build step. No bundler. No framework.

## Data

All metrics are real. Sourced from the hotel's property management system and financial records. Data as of November 2025.
